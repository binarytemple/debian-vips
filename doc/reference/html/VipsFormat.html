Object Hierarchy

Description

VIPS has a simple system for representing image load ans save operations in a generic way.

enum VipsFormatFlags

typedef enum {
	VIPS_FORMAT_NONE = 0,		/* No flags set */
	VIPS_FORMAT_PARTIAL = 1		/* Lazy read OK (eg. tiled tiff) */
